**Limits** are set according to the selected plan, restricting users, projects, clients, and instances.
<Note>You can increase any of these limits by clicking **"Request Limit"**, entering the desired amount, and submitting.</Note>

### Limit Types
- **Users**: Maximum number of users allowed in the system
- **Customers**: Maximum number of customers allowed in the account
- **Customer Users**: Maximum number of users allowed per customer account
- **Contacts**: Maximum number of contact records for your users
- **Projects**: Maximum number of active projects permitted
- **Instances**: Maximum number of system instances available for your usage
