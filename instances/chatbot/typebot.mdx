This guide provides detailed instructions for configuring the integration between Evolution Bot and Typebot.


## **Enabled**  
Activate or deactivate the Typebot integration.  
- **Enabled:** Allows communication between Evolution Bot and Typebot.  
- **Disabled:** Disables this integration.  

---

## **Description***  
A required field to briefly describe this integration instance.  
- Example: "Customer Support Bot for Typebot."  

---

## **Typebot Settings**  

### **Typebot API URL***  
The API endpoint URL for your Typebot instance.  
- Ensure that this URL points to a valid Typebot API.  

### **Public Name of the Typebot**  
The visible name associated with this Typebot integration.  
- This name helps identify the Typebot instance in your system.  

---

## **Trigger Settings**  

### **Trigger Type**  
Defines the condition that triggers the Typebot.  
- **Keyword:** The Typebot will be triggered by a specific keyword.  

### **Trigger Operator**  
Specifies the rule for matching the keyword in messages.  
- **Contains:** Trigger fires when the keyword is found within a message.  

### **Trigger**  
The keyword that activates the Typebot when used in a conversation.  
- Example: "Support."  

---

## **General Settings**  

### **Expire in Minutes**  
Sets the time in minutes after which the bot session expires.  
- **0:** No expiration.  

### **Keyword to Finish**  
Defines the keyword used to manually end a bot session.  
- Example: "Goodbye."  

### **Default Delay Message**  
The default delay time between bot messages in seconds.  
- **0:** No delay.  

### **Unknown Message**  
Handles responses when the bot receives an unknown message.  

### **Listen from me**  
Defines whether the bot listens to user messages when the sender is the bot itself.  

### **Stop Bot from Me**  
Allows stopping the bot when a specific action or condition is triggered.  

### **Keep Open**  
Keeps the conversation active, even after the bot finishes its interaction.  
- **Enabled:** Keeps the session open.  
- **Disabled:** Ends the session automatically.
