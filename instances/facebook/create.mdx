# Creating Facebook Instance

To create a Facebook instance, follow these steps:

## Configuration Fields

1. **Select Customer**  
   Choose the customer who will own this instance

2. **Instance Name**  
   Enter a name to identify your instance

3. **Channel Selection**  
   Select "Facebook" from the available channels

4. **Facebook Connection**  
   Click "Connect with Facebook" to authenticate and link your Facebook Page

<img src="/images/instance/create5.png" alt="Facebook Instance Creation" />

## Connection Process

When you click "Connect with Facebook":
1. A Facebook login popup will appear
2. Select the Facebook Page you want to connect
3. Review and accept the required permissions
4. The instance will be created and connected automatically

<Note>
  Make sure you have admin access to the Facebook Page you want to connect.
</Note>

## Configure Integrations

Once connected, you can set up:
- Chatbots for automated messaging
- Custom events for specific triggers
- API access using the instance token (apikey) and ID
