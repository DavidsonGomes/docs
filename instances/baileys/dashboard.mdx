The instance dashboard provides an overview of your instance where you can:
- View and update the instance name
- Monitor key metrics:
  - Number of contacts
  - Message count
  - Active chats
  - Instance status
- Manage instance settings
- Delete the instance if needed

For WhatsApp Web channels, use the **"Get QR Code"** button to connect your WhatsApp account to the instance.
