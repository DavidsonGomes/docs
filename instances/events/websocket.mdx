To enable and configure the websocket:
1. Click the enable button
2. Select the events you want to track
3. Use the following URL format in your system:
   ```
   wss://api.yourdomain.com/{instanceName}
   ```
   Replace `yourdomain.com` with your domain or IP if running Evolution on a separate server.

<Note>Make sure your system can maintain a persistent WebSocket connection for real-time updates.</Note>

Once configured, you'll receive messages triggered by the Evolution websocket.
