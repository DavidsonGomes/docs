# Instagram Dashboard

The Instagram instance dashboard provides comprehensive monitoring and management tools.

## Features

### Status Panel
- Connection status
- Webhook health
- Message queue status
- API quota usage

### Message Statistics
- Direct messages sent/received
- Story replies
- Message reactions
- Media shares

### Account Information
- Profile details
- Follower count
- Content metrics
- Engagement rates

## Management Tools

- **Quick Actions**
  - Refresh connection
  - Clear message queue
  - Update permissions
  - Test webhook

- **Monitoring**
  - API call logs
  - Error tracking
  - Performance metrics
  - Rate limit monitoring

<Note>
  Monitor your API usage to stay within Instagram's rate limits.
</Note>
