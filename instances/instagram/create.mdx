# Creating Instagram Instance

To create an Instagram instance, follow these steps:

## Configuration Fields

1. **Select Customer**  
   Choose the customer who will own this instance

2. **Instance Name**  
   Enter a name to identify your instance

3. **Channel Selection**  
   Select "Instagram" from the available channels

4. **Instagram Connection**  
   Click "Connect with Instagram" to authenticate and link your Instagram account

<img src="/images/instance/create6.png" alt="Instagram Instance Creation" />

## Connection Process

When you click "Connect with Instagram":
1. A Meta login popup will appear
2. Select the Instagram Business account you want to connect
3. Review and accept the required permissions
4. The instance will be created and connected automatically

<Note>
  Make sure you have a Business or Creator Instagram account and admin access to connect it.
</Note>

## Configure Integrations

Once connected, you can set up:
- Chatbots for automated messaging
- Custom events for specific triggers
- API access using the instance token (apikey) and ID
