To create an instance in Evolution, follow these steps:

1. **Select a Customer**
   - Choose from your project's existing customers
   - Ensure the customer has enabled channels

2. **Choose a Channel**
   - Select from the customer's available channels:
     - WhatsApp Web
     - WhatsApp API
     - Telegram
     - Other supported channels
   - Some channels may require additional information (e.g., phone number)

3. **Connect the Instance**
   
   For WhatsApp Web:
   - Generate and scan the QR code within the instance
   
   For other channels:
   - Provide the required configuration details

4. **Configure Integrations**
   
   Once connected, you can set up:
   - **Chatbots** for automated messaging
   - **Custom events** for specific triggers
   - **API access** using the instance token (`apikey`) and ID
